# ✨ Overview

AI ChatBot is a conversational AI application that combines the speed of **Groq's LPU inference** with the power of **Llama 3.3 70B** to provide fast, natural, and context-aware responses.

Designed as a lightweight yet scalable AI assistant, the project demonstrates modern AI integration, API communication, and web deployment using Gradio and Render.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/srjofficial/AI-ChatBot.git

cd AI-ChatBot
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure Environment Variable

Create an environment variable named

```
GROQ_API_KEY
```

and assign your Groq API key.

---

## Run

```bash
python app.py
```
